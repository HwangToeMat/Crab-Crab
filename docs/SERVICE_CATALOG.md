# 🦀 Crab-Crab AI-Native 서비스 카탈로그 (Total 20 Services)

꽃게팀 V2.0 '진화' 시스템을 통해 기획 및 구현된 20가지 핵심 AI 서비스 리스트입니다.

| 서비스명 | 핵심 가치 (Value Proposition) | 주요 기능 (Key Features) | 기술적 특징 |
| :--- | :--- | :--- | :--- |
| **1. Crab-Nexus** | **통합 대시보드** | 19개 서브 서비스의 통합 관리 및 진입점 | React/Vite 기반 중앙 제어 |
| **2. Mood-Ge** | **감정 케어** | 사용자 일기/대화 기반 정밀 감정 분석 | AI 지능형 로딩 애니메이션 |
| **3. Crab-Infinity** | **무한 창의성** | AI 기반 아이디어 발굴 및 마인드맵 확장 | 디자인 토큰 기반 무한 스크롤 |
| **4. God-Crab** | **종합 자문** | 범용적인 AI 비서 및 복합 문제 해결 | 고성능 Python 백엔드 연동 |
| **5. Crab-Shield** | **보안 관제** | 실시간 취약점 스캔 및 AI 공격 방어 | `global_validator` 연동 |
| **6. Crab-Finance** | **자산 관리** | AI 지능형 가계부 및 투자 패턴 분석 | 복합 데이터 시각화 |
| **7. Crab-Health** | **건강 분석** | 생활 패턴 기반 개인화 건강 가이드 | 비정형 데이터 정규화 |
| **8. Crab-Mate** | **소셜 네트워킹** | 취향 기반 AI 매칭 및 소통 어시스턴트 | 실시간 소켓 통신 최적화 |
| **9. Crab-Mind** | **심리 상담** | 전문 심리 분석 및 정서적 지지 제공 | 민감 정보 익명화 처리 |
| **10. Crab-Scan** | **문서 분석** | 문서 요약, 번역 및 핵심 정보 추출 | OCR 및 LLM 복합 연동 |
| **11. Crab-Sentinel** | **침입 탐지** | 네트워크 트래픽 분석 및 위협 경고 | 저지연 실시간 모니터링 |
| **12. Crab-Soul-Care**| **명상/수면** | 개인별 최적화된 명상 가이드 및 수면 분석 | 오디오 시각화 및 무드 제어 |
| **13. Crab-Link** | **네트워크 연결** | 다양한 서비스 간 API 브릿지 역할 | 이벤트 기반 아키텍처 |
| **14. Deep-Dream** | **창의적 시각화** | 텍스트 기반 이미지 생성 및 예술적 변환 | Stable Diffusion 등 AI 연동 |
| **15. Work-Crab** | **생산성 도구** | 태스크 관리, 자동 이메일 작성 및 업무 보조 | 칸반 보드 및 캘린더 자동화 |
| **16. Eco-Charge** | **에너지 최적화** | 에너지 사용 패턴 분석 및 절약 가이드 | IoT 데이터 수집 및 시뮬레이션 |
| **17. Nuri-Bom** | **다국어 소통** | 실시간 통번역 및 글로벌 커뮤니케이션 | Context 기반 정밀 번역 |
| **18. Soso-Haeng** | **일상의 행복** | 소소한 성취 기록 및 보상 시스템 | 게이미피케이션 요소 도입 |
| **19. SentiCrypto** | **크립토 분석** | 가상자산 시장 감성 분석 및 트렌드 예측 | 시장 데이터 실시간 스트리밍 |
| **20. Crab-Crab-Care**| **자가 진화 케어** | 시스템 자체의 헬스체크 및 성능 유지 관리 | 자가 리팩토링 및 튜닝 |
