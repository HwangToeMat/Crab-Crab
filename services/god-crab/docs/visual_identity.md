# Visual Identity: God-Crab (갓게)

## 1. Brand Concept
-   **Keywords**: Friendly, Energetic, Reliable, Cute.
-   **Visual Metaphor**: The Crab (hard shell for discipline, soft inside for kindness).

## 2. Color Palette (Design Tokens)
-   **Primary (Crab Orange)**: `#FF6B6B` - Representing energy and the Crab's shell.
-   **Secondary (Ocean Blue)**: `#4D96FF` - Stability and productivity.
-   **Success (Deep Sea Teal)**: `#06D6A0` - Completed tasks.
-   **Warning (Sand Yellow)**: `#FFD166` - Streaks and levels.
-   **Background**: `#F8F9FA` - Clean and simple for focus.

## 3. Typography
-   **Headings**: "Pretendard Variable", sans-serif (Bold, 24px+) - Modern and highly readable.
-   **Body**: "Pretendard Variable", sans-serif (Medium, 16px) - Clarity for tasks.

## 4. UI Components Strategy
-   **Crab Avatar**: Central visual element. Changes appearance based on level.
-   **Task Cards**: Rounded corners (12px), subtle shadows.
-   **Progress Bar**: Using the Primary Orange with a shell-like texture or pattern.

## 5. Tone & Voice
-   **Friendly & Encouraging**: "You're doing great, Master Crab!" or "Let's make today a Godsaeng day!"
