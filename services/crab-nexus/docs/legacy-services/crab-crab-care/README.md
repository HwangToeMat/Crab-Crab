# Crab-Crab Care (Silver Care AI Agent)

## 서비스 개요
**Crab-Crab Care**는 2026년 한국의 초고령사회 진입에 맞춰 기획된 실버 케어용 AI 에이전트 서비스입니다. 
v2에서는 실제 사용자(에이전트)들의 피드백을 바탕으로 접근성과 데이터 가시성을 대폭 개선하였습니다.

## 주요 기능
-   **v2: AI 음성 읽어주기 (TTS)**: 눈이 침침한 어르신들을 위해 AI의 답변을 소리로 들려주는 기능 추가.
-   **v2: 주간 건강 트렌드**: 오늘 하루뿐만 아니라 지난 7일간의 활동량(걸음수) 추이를 시각화하여 제공.
-   **v2: 보호자 알림 시스템**: 심박수 이상 등 긴급 상황 발생 시 보호자에게 자동으로 알림 전송 (Mock).
-   **AI 말벗 (Digital Grandchild)**: 고령 사용자와 친근하게 대화하며 정서적 지지 제공.
-   **실시간 건강 모니터링**: 심박수, 걸음 수, 수면 패턴 등 데이터 시각화.

## 기술 스택
-   **Backend**: Python, FastAPI
-   **Frontend**: HTML5, CSS3, Vanilla JavaScript (Web Speech API 사용)
-   **Communication**: REST API

## 버전 기록
-   **v1.0**: 초기 릴리스 (기본 채팅 및 대시보드)
-   **v2.0**: 사용자 피드백 반영 (TTS, 주간 트렌드, 보호자 알림 기능 추가)
