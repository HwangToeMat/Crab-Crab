# 🦀 Crab-Soul-Care (꽃게 소울 케어) PRD v1.0

## 1. 프로젝트 개요
- **서비스 명:** Crab-Soul-Care (꽃게 소울 케어)
- **한줄 설명:** Gemini AI 기반 감정 분석 및 초개인화 멘탈 웰니스 가이드.
- **핵심 가치:** "당신의 마음을 꽃게처럼 단단하고 부드럽게 감싸안습니다."

## 2. 주요 기능 (MVP)
### 2.1. Emotion Mirror (감정 분석)
- 사용자의 텍스트 입력 또는 음성(시뮬레이션)을 Gemini API로 분석하여 현재의 8가지 감정 상태 지표 도출.
- 분석 결과에 따라 맞춤형 테라피 추천.

### 2.2. Soul Soundscape (사운드 테라피)
- 감정 상태에 맞는 백색소음, ASMR, 명상 음악을 동적으로 생성 및 추천.
- '꽃게의 속삭임' (부드러운 해변 파도 소리 등) 시그니처 사운드 포함.

### 2.3. AI Wellness Coach
- 대화형 인터페이스를 통해 고민 상담 및 정서적 지지 제공.
- 10회 검증 루프를 통해 상담 품질 고도화.

## 3. 기술 스택
- **Backend:** Python (FastAPI) - 고성능 비동기 API 처리 및 Gemini 연동.
- **Frontend:** HTML5 / Vanilla CSS / JS (Responsive Dark/Soft UI).
- **AI Engine:** Google Gemini API - 정교한 감정 및 심리 분석.
- **Infrastructure:** Docker & Docker-Compose.

## 4. 디자인 컨셉

## 5. 로드맵
- **Phase 1:** 기획 및 PRD 확정 (완료)
- **Phase 2:** 백엔드(감정 분석기) 및 프론트엔드(명상 대시보드) 개발
- **Phase 3:** 10회 검증 루프 가동 및 상담 엔진 고도화
- **Phase 4:** 최종 리포트 및 메인 브랜치 통합
