# 누리봄 (Nuri-Bom)

1.5 가구를 위한 하이퍼로컬 미션 수행 및 느슨한 연대 플랫폼

## 🌸 서비스 컨셉
"이웃과 함께하는 따뜻한 봄날 같은 일상"
독립적인 삶을 존중하면서도 필요할 때 따뜻하게 연결되는 하이퍼로컬 커뮤니티입니다.

## ✨ 주요 기능
- **마이크로 미션:** 피자 나눠먹기, 산책 함께하기, 공구 빌리기 등 사소하지만 확실한 도움 주고받기
- **AI 누리:** 사용자의 성향과 동네 트렌드를 분석하여 최적의 미션 추천
- **실시간 채팅:** 미션 수행을 위한 안전하고 빠른 소통창구

## 🛠 기술 스택
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js (Express), Socket.io
- **Database:** SQLite

## 🚀 실행 방법
### Backend
```bash
cd services/nuri-bom/backend
npm install
node main.js
```

### Frontend
- `services/nuri-bom/frontend/index.html` 파일을 브라우저에서 엽니다.
