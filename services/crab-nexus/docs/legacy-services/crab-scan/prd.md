# Product Requirement Document (PRD): Crab-Scan (크랩스캔)

## 1. Service Overview
- **Service Name**: Crab-Scan (크랩스캔)
- **Tagline**: "AI-driven Godsaeng Spending: Save more with the Smart Crab."
- **Core Value**: AI-based receipt analysis and frugal living advice to combat inflation and promote disciplined spending habits.

## 2. Target Audience
- Young professionals and students (20s-30s) who want to manage their expenses with a fun, gamified AI assistant.
- Users interested in "Frugal Tech" (짠테크) and "Godsaeng" (갓생) lifestyle.

## 3. Key Features (Phase 1-4)
### 3.1 AI Receipt Analyzer (Gemini API Based)
- Users input text from a receipt (OCR simulation).
- AI analyzes the spending category (Food, Luxury, Utility, etc.).
- AI provides "Crab Advice" (e.g., "Drinking a expensive latte? That's a 'Shell-breaking' expense!").

### 3.2 Crab Spending Dashboard
- Visualization of daily/weekly spending.
- "Saving Score" (0-100) based on AI feedback.

### 3.3 Frugal Challenges (Crab Quests)
- Daily challenges like "No-spend Day" or "Home-cooked Meal Challenge."
- Completing quests earns "Crab Shells" (Tokens).

### 3.4 Interactive UI
- Deep Blue & Electric Purple theme (Infinite Evolution).
- Interactive Crab animations reacting to spending scores.

## 4. Technical Architecture
- **Backend**: Python (FastAPI) for lightweight and fast API handling.
- **Frontend**: Vanilla JS, CSS (Modern Layout) for high performance.
- **AI Engine**: Gemini API for sentiment and spending analysis.
- **Deployment**: Docker-ready environment.
