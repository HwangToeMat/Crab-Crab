# 🥗 Crab-Health (꽃게헬스) PRD

## 1. 프로젝트 개요
- **서비스명:** Crab-Health (꽃게헬스)
- **한줄 설명:** AI 기반 개인형 맞춤 운동 루틴 추천 및 식단 코칭 서비스
- **핵심 가치:** "딱딱한 껍질처럼 단단한 근육과 건강을 꽃게팀이 만들어 드립니다."

## 2. 주요 기능
- **AI 맞춤형 운동 루틴:** 사용자의 체력 수준과 목표에 따라 AI가 매일 새로운 운동 세트 제안.
- **지능형 식단 분석:** 섭취한 음식을 입력하면 AI가 영양 균형도를 평가하고 개선점 제시.
- **꽃게 코칭 메시지:** 사용자의 성취도에 따라 꽃게 캐릭터가 재미있고 직관적인 피드백 제공.

## 3. 기술 스택
- **Backend:** Python (FastAPI)
- **Frontend:** HTML5, Vanilla CSS, JS (Interactive Health Dashboard)
- **AI Engine:** Google Gemini API (Personalized Health Consulting)
- **Infrastructure:** Docker, docker-compose

## 4. 로드맵
- **Phase 1-2:** 운동/식단 로직 및 대시보드 UI 개발
- **Phase 3-4:** 10회 검증 루프 (데이터 정합성 및 UI 반응성 확인)
- **Phase 5:** 실시간 운동 영상 분석 연동 (향후 진화 단계)
