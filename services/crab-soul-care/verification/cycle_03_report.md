# Cycle 03 Report - Keyword-based Emotion Analysis

## 개선 사항
- `backend/main.py`에 키워드 기반 감정 분석 로직 도입.
- `KEYWORD_MAP`을 정의하여 입력된 텍스트에서 감정을 추출하는 룰 베이스 엔진 구현.
- 매칭되는 키워드가 없을 경우를 대비한 하이브리드(키워드+랜덤) 분석 방식 적용.

## 검증 결과
- "너무 행복해" 입력 시 'Joy' 테마로, "요즘 일이 많아 피곤해" 입력 시 'Stress' 테마로 정확히 분석됨을 확인.
- 키워드 매칭 시 신뢰도(confidence) 점수가 더 높게 반영됨을 확인.

## 향후 계획
