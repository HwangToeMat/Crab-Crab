# Product Requirement Document (PRD): Mood-Ge (무드게)

## 1. Service Overview
- **Service Name**: Mood-Ge (무드게)
- **Tagline**: "Burnout recovery, one small crab at a time."
- **Core Value**: Helping users overcome burnout through small daily achievements and emotional awareness.

## 2. Target Audience
- Busy professionals and students in Korea feeling emotionally drained (Burnout).
- Users looking for a simple, cute, and gamified way to track habits and moods.

## 3. Key Features (MVP)
### 3.1 Daily Mood Tracker
- Users select an emoji/feeling for the day.
- A "Mood Crab" character changes its color or expression based on the selection.

### 3.2 Micro-Challenge (The "Crab Quest")
- 3 random simple tasks provided daily (e.g., 'Drink 1 cup of water', 'Write 1 sentence of gratitude').
- Completing tasks earns "Crab Points."

### 3.3 Emotional Dashboard
- Weekly/Monthly visualization of mood trends.
- "Burnout Recovery Level" indicator based on consistency.

### 3.4 Interactive Crab Home
- A virtual space where the user's Crab lives, improving as tasks are completed.

## 4. Technical Stack (Proposed)
- **Frontend**: React (SPA) with Vanilla CSS (per guidelines).
- **Backend**: Python (FastAPI) for simple API interactions.
- **Storage**: Simple JSON-based storage or SQLite for local prototype.

## 5. Success Metrics
- Daily Active Users (DAU).
- Average Number of Challenges Completed per user.
- User sentiment improve through the "Burnout Recovery Level" metric.
