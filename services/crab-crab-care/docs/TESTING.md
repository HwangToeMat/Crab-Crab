# Crab-Crab Care 테스트 가이드

이 가이드는 로컬 환경에서 **Crab-Crab Care** 서비스를 실행하고 테스트하는 방법을 설명합니다.

## 1. 사전 준비
-   **Python 3.8+**가 설치되어 있어야 합니다.
-   필요한 패키지를 설치합니다:
    ```bash
    pip install fastapi uvicorn pydantic python-dotenv
    ```

## 2. 백엔드 서버 실행
`services/crab-crab-care/backend` 디렉토리로 이동하여 서버를 실행합니다:
```bash
python main.py
```
서버는 기본적으로 `http://localhost:8000`에서 실행됩니다.

## 3. 프런트엔드 실행
`services/crab-crab-care/frontend` 디렉토리에 있는 `index.html` 파일을 브라우저(Chrome, Edge 등)에서 직접 엽니다.

## 4. 테스트 시나리오

### 시나리오 1: AI 말벗 대화
1.  하단의 입력창에 "오늘 기분이 좀 적적하네"라고 입력하고 [말하기] 버튼을 누릅니다.
2.  AI(Crab-Care)가 친근한 말투로 응답하는지 확인합니다.

### 시나리오 2: 실시간 건강 데이터 확인
1.  상단의 '오늘의 건강' 카드에서 심박수, 걸음수, 수면시간이 표시되는지 확인합니다.
2.  데이터는 10초마다 서버에서 새로운 값을 가져와 갱신됩니다.

### 시나리오 3: 상태 알림 확인
1.  우측의 '알림' 카드에 현재 필요한 복약 알림이나 활동 권장 메시지가 뜨는지 확인합니다.

### 시나리오 4: API 직접 테스트 (Swagger UI)
1.  브라우저에서 `http://localhost:8000/docs`에 접속합니다.
2.  FastAPI가 제공하는 자동 문서화 화면에서 `/chat`, `/health-status`, `/alerts` 엔드포인트를 직접 실행해 볼 수 있습니다.

## 5. 문제 해결
-   **API 호출 실패**: 백엔드 서버가 8000 포트에서 정상 실행 중인지 확인하세요.
-   **한글 깨짐**: 브라우저 인코딩 설정이 UTF-8인지 확인하세요 (기본적으로 지원됩니다).
