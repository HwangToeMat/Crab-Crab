# Crab-Crab Care (Silver Care AI Agent)

## 서비스 개요
**Crab-Crab Care**는 2026년 한국의 초고령사회 진입에 맞춰 기획된 실버 케어용 AI 에이전트 서비스입니다. 
고령 사용자의 건강 데이터를 실시간 모니터링하고, AI 말벗 기능을 통해 정서적 유대감을 형성하며, 필요한 복약 알림 및 일정 관리를 자율적으로 수행합니다.

## 주요 기능
-   **AI 말벗 (Digital Grandchild)**: 고령 사용자와 친근하게 대화하며 정서적 지지 제공.
-   **실시간 건강 모니터링**: 심박수, 걸음 수, 수면 패턴 등 데이터 시각화 (Mock Data).
-   **지능형 알림**: 복약 시간, 활동 부족 알림 등 생활 밀착형 관리.
-   **고대비 UI**: 시력이 약한 어르신들을 위한 크고 명확한 디자인.

## 기술 스택
-   **Backend**: Python, FastAPI
-   **Frontend**: HTML5, CSS3, Vanilla JavaScript
-   **Communication**: REST API

## 디렉토리 구조
-   `backend/`: FastAPI 기반 서버 로직 및 API 엔드포인트.
-   `frontend/`: 사용자 웹 인터페이스.
-   `docs/`: 서비스 관련 문서 및 테스트 가이드.
