# 💰 Crab-Finance (꽃게머니) PRD

## 1. 프로젝트 개요
- **서비스명:** Crab-Finance (꽃게머니)
- **한줄 설명:** AI 기반 지능형 지출 분석 및 맞춤형 자산 관리 솔루션
- **핵심 가치:** "당신의 돈이 새나가지 않도록 꽃게 집게처럼 꽉 잡아드립니다."

## 2. 주요 기능
- **AI 자동 지출 분류:** 소비 내역을 텍스트로 입력하면 AI가 카테고리(식비, 주거, 취미 등) 자동 분류.
- **스마트 소비 레포트:** 시각화된 차트를 통해 월간 소비 패턴 분석 및 전월 대비 변동폭 제공.
- **꽃게 저축 챌린지:** 목표 금액 설정 시 AI가 일일 권장 소비액 가이드 제공.

## 3. 기술 스택
- **Backend:** Python (FastAPI)
- **Frontend:** Vanilla JS + CSS (Responsive Dashbord), Chart.js
- **AI Engine:** Google Gemini API (Financial Insight Generation)
- **Database:** SQLite (Local Data Storage for Prototype)

## 4. 로드맵
- **Phase 1-2:** 데이터 스키마 설계 및 대시보드 UI 개발
- **Phase 3-4:** 10회 검증 루프 (데이터 정합성 및 차트 렌더링 확인)
- **Phase 5:** AI 금융 비서 연동 (지출 패턴 기반 절약 팁 자동 생성)
