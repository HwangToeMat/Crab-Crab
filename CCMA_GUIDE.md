# Crab-Crab Multi-Agent (CCMA) System Guide

Welcome to the **CCMA (Crab-Crab Multi-Agent)** system. This framework is designed to orchestrate specialized agents to collaboratively build and verify AI-driven IT services.

## System Architecture

-   **agents/**: Persona definitions for builders and critics.
-   **tasks/**: Tracking task status and data.
-   **workflows/**: Interaction logic (e.g., `orchestrator.py`).
-   **outputs/**: Final artifacts (PRDs, code, reports).

## The CCMA Ecosystem v2.0

### 1. CrabTeam (Builders)
-   **Core**: Trend Analyst, PM, UI/UX, Tech Architect, Frontend/Backend Dev, DevOps, QA.
-   **V2 Specialization**:
    -   **AI Integration**: Utilizing Gemini API for intelligent features.
    -   **Trust Systems**: Designing 'Crab Warranty' and 'Ambassador' models.

### 2. Evaluation Group (Critics)
-   **User Persona Group**: 10 diverse critics (Age 10s to 60s).
-   **Consumer Insight Expert**: Translates feedback into improvement roadmaps.

## Mutual Verification & AI Loop

### Standard Verification:
1.  **Drafting** -> **Verification** -> **Feedback** -> **Approval**.

### Advanced AI Evolution Loop:
-   **Persona Simulation**: Testing features against 10 specific user personas.
-   **Feedback Synthesis**: Aggregating insights to drive the next sprint.
-   **Evolutionary Iteration**: Continuous scaling through **"꽃게팀 진화개시"**.

## Development Standards (V2.0)
-   **Tech Stack**: FastAPI (Python), Vanilla JS/HTML/CSS (Lightweight), SQLite/SQLAlchemy.
-   **AI Engine**: Gemini-powered recommendation and assistant modules.
-   **Trust Layer**: Mandatory implementation of 'Crab Temperature' and 'Warranty' systems.
-   **Visuals**: Use 'Crab Orange' as Primary, 'AI Electric Purple' as Secondary color for intelligence features.

## How to Run
-   Initiate Service: `꽃게팀 작업개시`
-   Evolve Service: `꽃게팀 진화개시`
